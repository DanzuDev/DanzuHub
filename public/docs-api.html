Dokumentasi API DanzuHub

Selamat datang di dokumentasi API DanzuHub. Anda dapat mengunggah file langsung ke DanzuHub menggunakan curl atau melalui endpoint API dengan bot Anda.

  • API unggah file sederhana dengan respons URL langsung
  • Tidak memerlukan autentikasi (API publik)
  • Waktu respons cepat
  • URL yang rapi (danzuhub.my.id/u/[namafile])

üîπ Endpoint API

POST https://danzuhub.my.id/api/upload

üîπ Parameter Request

  • file wajib - File yang akan diunggah (mendukung semua jenis file umum)

üîπ Contoh Request (cURL)

curl -F "file=@contoh.jpg" https://danzuhub.my.id/api/upload

üîπ Respons Berhasil

  "success": true,
  "url": "https://danzuhub.my.id/u/abc123.jpg",
  "filename": "abc123.jpg",
  "size": 1024,
  "timestamp": 1620000000
}

üîπ Respons Gagal

{
  "success": false,
  "message": "Gagal mengunggah file",
  "error": "Ukuran file melebihi batas",

Batas Penggunaan

Saat ini tidak ada batasan ketat, tapi harap gunakan dengan bijak. Request berlebihan mungkin akan dibatasi sementara.

Pembatasan File

  • Ukuran maksimal file: 200MB
  • Jenis file yang didukung: Gambar, video, dokumen, dan arsip umum
  • File disimpan selamanya
